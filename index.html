School District App

Legend Web Works

What is it?

This was a project given to me after about 6 months of implementing responsive, and non-responsive websites. The goal was to find a way to showcase critical school information in an accessible, intuitive, and quick way.

We determined what information should be featured most prominently in the app model by using information gathered from Google Analytics on existing schools sites. Also, we incorporate each client’s preferences/needs for other content.

How's it work?

  • Mobile traffic is redirected to a subdomain that uses the same database as the main desktop site, but formats the content in an entirely different way.
  • When first arriving on the mobile-only subdomain, users are asked to choose preferences for their schools. The chosen schools and the corresponding news pages are saved into a cookie and we are able to modify the content we load on each page based on those cookies.
  • This app model also features a manifest.json file, service workers for caching, and saving to home screen.
  • Technologies used: PHP, HTML, CSS (boostrap 3), JS, jQuery, MySQL

What?

Similar to the app above but with more AJAX, this app loads content from the main site’s database.
