Progressive Web Apps

School District Web App

Legend Web Works

What is it?

I was tasked to find a way to showcase critical school information in an accessible and intuitive mobile format. I achieved this by using the main site's database and formatting the content to be responsive. The team worked together to determine what information should be featured most prominently in the app model by using information gathered from Google Analytics on existing schools sites.

How does it work?

  • Mobile traffic is redirected to a subdomain that uses the same database as the main desktop site.
  • When first arriving on the mobile-only subdomain, users are asked to choose preferences for their schools. The chosen schools and the corresponding news pages are saved into a cookie and we are able to modify the content we load on each page based on those cookies.
  • This web app model also features a manifest.json file, service workers for caching, and saving to home screen.
  • Technologies used: HTML, CSS, Bootstrap 3, PHP, JS, jQuery, MySQL

Athletics Department Web App

Legend Web Works

What is it?

School athletics departments use a different CMS than the schools. Similar to the schools above, I was tasked to find a way to showcase critical information in a mobile format regarding school sports. The design of this app was directly influenced by the industry leader's mobile app (ESPN).

How does it work?

  • Similar to the school app above above but with more asynchronous features, this app loads content from the main site’s database. Users are forwarded to the subdomain using a PHP Mobile Detect class.
  • This model features a manifest.json and service worker file that prompt users to download to their home screen and cache the data for (limited) offline viewing.
  • Technologies used: HTML, CSS, Bootstrap 3, PHP, JS, jQuery, MySQL

What was my role?

This ecommerce module was originally developed within a non-responsive front end. In 2018, I was tasked to take all of the existing functionality and make it responsive. By doing research on what top ecommerce companies were doing, I was able to create an intuitive, modern, and familiar ecommerce experience.

Updated Features Include:

  • Fully responsive cart & checkout
  • Option to display items in a "table" that turns into mobile friendly layout on mobile devices
